welcome to my world.

## Set Up

### Initialise
```sh
npm i
```
### Initialise Houdini
```sh
npm init:houdini
```

### Generate Houdini types or whatever..idk
```sh
npm generate:houdini
```

## Run It

```sh
npm run launch
```

## Build It

```sh
npm run build
```

## Preview

```sh
npm run preview
```
